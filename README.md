# 🎮 Task 1 — Hangman Game

## About

A console-based Hangman Game developed in Python where players must guess a hidden word before running out of attempts.

## Features

- Multiple Categories

  - Animals
  - Sports
  - Science
- Difficulty Levels
- Hint System
- Score Calculation
- Player Statistics
- Achievement System
