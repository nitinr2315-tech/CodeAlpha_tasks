# 🎮 Hangman Game in Python

## 📌 Project Overview

This is a console-based Hangman Game developed in Python. The player has to guess a hidden word one letter at a time before the maximum number of attempts is exhausted.

The project includes multiple categories, difficulty levels, hints, scoring system, leaderboard, player profile, statistics, and achievements.

## 🎯 Game Rules

1. Guess one letter at a time.
2. Each wrong guess reduces available attempts.
3. A hint can be used once per game.
4. Correct guess earns points.
5. Wrong guess reduces score.
6. Guess the complete word before attempts run out.

---

## 🏆 Scoring System

| Action        | Score |
| ------------- | ----- |
| Correct Guess | +10   |
| Wrong Guess   | -2    |
| Hint Used     | -5    |
