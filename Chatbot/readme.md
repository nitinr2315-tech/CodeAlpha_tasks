# 🤖 Basic Chatbot

A simple rule-based chatbot built using Python that responds to predefined user inputs and demonstrates core programming concepts.

## 🚀 Features

* Responds to greetings like **hello** and **hi**
* Answers simple questions
* Tells programming jokes
* Handles unknown inputs gracefully
* Exits with **bye**, **quit**, or **exit**
* Uses emojis for a friendly experience

## 🛠 Technologies Used

* Python 3

## 📂 Project Structure

```text
Basic-Chatbot/
│
├── chatbot.py

```

## ▶️ How to Run

```bash
python chatbot.py
```

## 💬 Example

```text
You: hello
Bot: 👋 Hi!

You: how are you
Bot: 😄 I'm fine, thanks!

You: bye
Bot: 👋 Goodbye! Have a great day!
```
